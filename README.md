## 📌 Sobre o Projeto

O **IRONCLAD Industrial OS** é uma plataforma moderna projetada para centralizar as operações industriais. Ele oferece uma interface robusta para monitorizar o desempenho da produção em tempo real, gerir tarefas das linhas de produção, controlar acessos da equipa e facilitar a comunicação através de um chat integrado. 

## ✨ Principais Funcionalidades

* **📊 Dashboard de Produção:** Monitorização em tempo real do desempenho da produção, cálculo de eficiência, saídas totais e métricas de qualidade com gráficos dinâmicos.
* **📋 Quadro de Produção (Kanban/Gantt):** Controlo diário de resultados através de quadros Kanban, permitindo a visualização de tarefas pendentes, em progresso, em revisão de qualidade (QA) e concluídas.
* **👥 Gestão de Equipas:** Lista detalhada de membros com níveis de acesso (Admin, Franchise, Operator), estado da rede e monitorização de *uptime*.
* **💬 Centro de Comando e Comunicação:** Sistema de chat integrado com suporte para canais, mensagens diretas, alertas automáticos do sistema e partilha de ficheiros importantes (Intelligence de Canais).

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido com tecnologias modernas para garantir alta performance e manutenibilidade:

* **[React 19](https://react.dev/)** - Biblioteca principal para a interface de utilizador.
* **[TypeScript](https://www.typescriptlang.org/)** - Tipagem estática para um código mais seguro.
* **[Vite](https://vitejs.dev/)** - Ferramenta de *build* e servidor de desenvolvimento ultra-rápido.
* **[Tailwind CSS v4](https://tailwindcss.com/)** - *Framework* de CSS baseada em utilitários para estilização rápida.
* **[Zustand](https://zustand-demo.pmnd.rs/)** - Gestão de estado global simples e leve.
* **[Recharts](https://recharts.org/)** - Biblioteca de gráficos baseada em componentes React.
* **[React Router DOM](https://reactrouter.com/)** - Navegação e roteamento da aplicação.
* **[Lucide React](https://lucide.dev/)** - Ícones consistentes e limpos.
